# V4 vs V5 — Management Comparison Report

> **V4** = Previous system &nbsp;|&nbsp; **V5** = Upgraded system (current)

---

## What Is This System?

An automated B2B lead generation workflow that:

1. Finds target companies matching client criteria
2. Qualifies them by checking ownership, services, and fit
3. Finds the right decision-maker contacts
4. Exports everything Apollo-ready for outreach — automatically

---

## What Changed from V4 to V5

| Area | V4 | V5 | Improvement |
|---|---|---|---|
| **Setup** | All questions asked together, no validation before saving | Questions asked one at a time, live filter test runs before saving | Catches wrong settings before campaign starts |
| **Company Search** | 2 search strategies, 25-company preview each | 3 search strategies, 5-company preview each | More options, 80% cheaper preview |
| **Qualification** | Pays to enrich all 100 companies upfront | Screens via free website scrape first, enriches only the ~60 that pass | Same result, fewer wasted credits |
| **Company Data** | Full data on all companies | Full data on qualified companies + website-scraped services detail | Richer service descriptions |
| **Contacts** | Phone for all contacts | Phone for top 2 contacts, past employment history added free | More useful contact profiles |
| **Export File** | Manual fix needed if format errors found | Auto-fixes formatting before generating CSV | Fewer Apollo import errors |
| **Campaign Resume** | Restart from beginning of a stage | Resume from the exact company where it stopped | No lost progress |

## Cost Comparison

> Crustdata credits = paid API calls to the data platform. Lower = better.

| Stage | V4 Credits | V5 Credits | Saving |
|---|---|---|---|
| Campaign setup | ~5 | ~5 | — |
| Company search preview | ~2 | ~0.5 | **~1.5 saved** |
| Company search (100 results) | ~3 | ~3 | — |
| Company qualification | ~200 | ~110 | **~90 saved** |
| Contact search | ~6 | ~6 | — |
| Contact enrichment | ~350 | ~260 | **~90 saved** |
| **Total per campaign** | **~566** | **~385** | **~181 saved (32%)** |

### Why V5 Costs Less

V4 pays to enrich all 100 companies — including ones that turn out to be wrong fits
(software companies, acquired firms, publicly traded). V5 first screens companies
using free website scraping, removes the obvious misfits (~40 companies), and
only pays for enrichment on the ~60 worth investigating.

## Advantages at a Glance

### V5 Wins

- **32% cheaper** per campaign on average
- More accurate company descriptions — pulled from actual website, not just database tags
- Ownership evidence includes a verbatim quote from the company's own page
- Contacts ranked 1 / 2 / 3 so sales team knows who to call first
- Auto-generated outreach note per contact saves manual writing time
- Past employer history on contacts at zero extra credit cost
- Campaign can resume at exact company level — no restart needed after interruption
- Live filter test catches setup errors before the campaign wastes any credits

### V4 Still Leads On

- Phone number for 3rd-ranked contact (V5 provides email only for 3rd contact)
- All-in-one conversational setup feels slightly faster to fill out

## Bottom Line

> V5 produces **equivalent or better lead quality** at **32% lower cost**.
> The only trade-off is no phone number on the 3rd-ranked contact per company —
> a minor gap since 3rd contacts are backup outreach where email is standard.

**Recommendation: Use V5 as the default for all new campaigns.**
